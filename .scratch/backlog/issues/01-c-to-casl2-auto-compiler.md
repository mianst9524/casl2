# C → CASL2 自動コンパイラの検討

Status: needs-triage

## 概要

C 言語対応は当面「キュレーション対訳例」（人手で用意した C ↔ CASL2 の対応）で実現する（ADR ではなく現時点の方針）。
将来、C のサブセットを CASL2 へ自動コンパイルする機能を追加するかどうかを残課題として記録する。

## 論点

- 対象とする C サブセットの範囲（代入・if・while・配列・関数まで等）
- 出力 CASL2 の読みやすさ（教材として読める質を保てるか）
- 実装規模（本物のコンパイラ＝字句解析・構文解析・コード生成）と保守コスト
- キュレーション対訳との役割分担（自動生成は近似、対訳は正確）

## 備考

「作らない」と決定したわけではない。当面はキュレーション方式で進め、需要と余力を見て再評価する。
